# Dooeats Website Walkthrough

## Customer Journey
1. **Landing Page**: Enter location to find nearby restaurants.
2. **Restaurant Listing**: Filter by category, rating, or delivery time.
3. **Menu Browsing**: Select a restaurant to view items.
4. **Cart**: Add items, customize options (toppings, size).
5. **Checkout**:
    - Select Delivery Address.
    - Choose Payment Method.
    - Apply Coupons (if any).
6. **Order Tracking**: Real-time status updates and driver location.

## Restaurant Partner Journey
1. **Dashboard**: Overview of daily sales and active orders.
2. **Menu Management**: Add/Edit categories, items, and modifiers.
3. **Order Management**:
    - **New**: Accept/Reject incoming orders.
    - **Processing**: Mark as "Ready" when preparation is done.
4. **Settings**: Configure operating hours, delivery zones, and payout details.

## Admin Journey
1. **Global Dashboard**: Platform-wide metrics (Total Orders, Revenue, Users).
2. **User Management**: View/Ban users, manage roles.
3. **Restaurant Approval**: Review and approve new restaurant signups.
4. **Finance**: View commission reports and process payouts.
5. **CMS**: Manage static pages, banners, and promotions.
